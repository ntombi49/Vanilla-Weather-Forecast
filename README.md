# 🌤 Weather Forecast App

This is a weather forecast web application built using HTML, CSS, and JavaScript.

## Features
- Search for any city
- Displays 5-day weather forecast
- Shows temperature, weather description, wind speed, and icons
- Dynamic UI colors based on weather conditions

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
